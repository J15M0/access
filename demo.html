About This Widget

The accessibility widget provides 12 different features organized in a 2-column grid. Each feature can be toggled by clicking its button in the panel. Some features cycle through multiple modes — watch the dots beneath the button to see the current level.

"The power of the Web is in its universality. Access by everyone regardless of disability is an essential aspect." — Tim Berners-Lee
